# E-Waste Nigeria

Standalone DIID article app for:

- `source/e-waste-generation-in-nigeria.docx`

This app visualises Nigeria's e-waste burden as a digital dependency story, with sections for:

- scale and continental ranking
- dependency loop
- health and environmental harm
- regulatory landscape
- rights implications
- policy recommendations
- data limitations

Local development:

- `npm run dev --workspace @diid/e-waste-nigeria`

Build:

- `npm run build --workspace @diid/e-waste-nigeria`
