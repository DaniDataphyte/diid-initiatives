# Child Online Safety

Standalone static policy app for:

- `Child Online Safety Legislation Across Africa`

## Build

```bash
npm run build:child-online-safety
```

## Output

- `dist/`

## Contents

- `index.html` static app shell
- `src/data/*` article datasets
- `src/main.js` article-specific interactive logic
- `src/styles/*` app-specific styling
- `source/brief.docx` source brief
